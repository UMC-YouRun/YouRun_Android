# 🏃 **유런(YOU+RUN)**
만나지 않아도 함께 뛰어요.

비대면 러닝메이트 챌린지 서비스, **유런(YOU+RUN)**<br><br>

## ❓ **About YOU+RUN**
친목 위주의 러닝 크루에 대한 입장 장벽과, 대면으로 같이 진행하는 러닝에 대한 거부감을 해소시켜줄 수 있으면 좋겠어!

이 두 문제를 해결하고, 러닝에 대한 동기부여까지 줄 수 있는,

**직접 만나지 않고도, 함께 뛰는 느낌을 주는 러닝 서비스**입니다.<br><br>

## 💻 **Tech stack**
### Libraries
- Kakao Map API<br>
→ 사용자의 위치를 지도상에 표시하고, 다양한 지도 관련 기능을 제공
- Lottie Animation<br>
→ 온보딩 화면의 애니메이션 효과<br><br>
이후 개발하면서 추가 예정...
### Frameworks and Tools
- [MVVM](https://github.com/UMC-YouRun/YouRun_Android/wiki/mvvm-pattern) Architecture
<br>→ 애플리케이션의 UI와 비즈니스 로직을 분리하여 보다 유지보수가 용이하고 확장 가능한 구조
- Location API
<br>→ Location Manager 또는 Google Play 서비스의 Location API를 사용하여 사용자의 위치 정보를 얻고 관리<br><br>
